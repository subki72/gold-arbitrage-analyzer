**Gold Arbitrage Analyzer** is a powerful, real-time web application designed to monitor and compare the prices of gold-backed cryptocurrency tokens (such as **PAXG** and **XAUT**) across major Centralized Exchanges (CEXs). 

By aggregating live market data from top-tier exchanges like Binance, OKX, Bybit, KuCoin, and others, this tool automatically calculates price spreads and identifies the most lucrative arbitrage opportunities. Whether you are a crypto trader, a data analyst, or a gold investor, the Gold Arbitrage Analyzer provides comprehensive data visualization—including price comparison charts, spread heatmaps, and historical price trends—empowering you to make data-driven decisions in real-time.

#### Key Features
* **Real-Time Data Aggregation:** Fetches live bid, ask, last price, and 24h volume data from 7+ major exchanges.
* **Automated Arbitrage Detection:** Instantly identifies the lowest (best buy) and highest (best sell) prices across all supported platforms to calculate potential profit margins.
* **Interactive Data Visualization:** Features dynamic price comparison charts, spread matrix heatmaps, and historical price trends.
* **Premium User Interface:** Built with a modern, responsive design and a default dark mode tailored for the best trading platform experience.
* **Reliable Benchmarking:** Integrates spot gold prices (XAU/USD) as a reliable benchmark against digital gold tokens.

#### Supported Tokens
* **PAXG / USDT** (Pax Gold)
* **XAUT / USDT** (Tether Gold)
